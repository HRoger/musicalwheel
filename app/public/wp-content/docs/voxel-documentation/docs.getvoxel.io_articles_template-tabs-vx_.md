## Template tabs (VX)

Notice: This is a legacy widget that is now deprecated and no longer recommended. The widget will be unlisted in the future versions of Voxel.

When creating tabbed layouts, we recommend using the elementor [Tabs widget](https://elementor.com/help/tabs-widget/) instead
