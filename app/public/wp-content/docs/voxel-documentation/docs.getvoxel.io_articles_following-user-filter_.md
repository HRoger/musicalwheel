## Following user filter

This filter is used to show Profiles who follow a specific user.

## Default values

The filter requires a user ID as value. This can be a static ID or a dynamic tag, for example:

- **Author > ID tag:** to show followers of the post author or profile author
- **User > ID tag:** to show followers of current user

## Options

- Label
- Key
- Icon
