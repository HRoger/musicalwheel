## Open now filter

Open now filter is used to filter posts by their open status based on [Work Hours field](docs.getvoxel.io_articles_work-hours-field_.md)

## Options

- Label
- Placeholder
- Key
- Data source: Requires a Work Hours field
- Match based on:

1\. Local time: The current time of each post based on the individual post timezone

2\. Site time: The current time based on the site timezone
- Description
- Icon

- Inline Switcher
- Button

## Default values

1. Checked
2. Unchecked
