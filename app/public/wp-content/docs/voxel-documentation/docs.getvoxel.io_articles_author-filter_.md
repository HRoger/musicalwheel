## Author filter

Author filter is used to filter posts by a specific author. The filter requires a user ID as default value, it has no effect without one provided.

## Options

- Label
- Key
- Icon

## Default values

The filter requires a user ID as default value.

That can be a static user ID, or a dynamic tag, for example User > ID for current user, or Author > ID for author of the current post.
