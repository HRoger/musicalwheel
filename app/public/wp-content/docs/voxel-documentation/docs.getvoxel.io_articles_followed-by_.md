## Followed by filter

This filter is used to show Profiles or CPTs which are followed by a specific user

## Default values

The filter requires a user ID as value. This can be a static ID or a dynamic tag, for example:

- **Author > ID tag:** to show Profiles or CPTs the post author or profile author follows
- **User > ID tag:**Â to show Profiles or CPTs the current user follows

- Key
