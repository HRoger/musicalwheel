## Switcher filter

Switcher filter is used to filter posts by the value of a [switcher field](docs.getvoxel.io_articles_switcher-field_.md)

## Options

- Label
- Placeholder
- Key
- Data source: Requires a Switcher field
- Comparison:

1\. Is checked

2\. Is unchecked
- Description
- Icon
